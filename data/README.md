## raw/

### Downloading IPEDS data

1. Go to [https://nces.ed.gov/ipeds/use-the-data/download-access-database](https://nces.ed.gov/ipeds/use-the-data/download-access-database)
2. Download the complete data files (CSV format) for survey years **2015–2023**
3. Extract each year's ZIP into `data/raw/ipeds/` — one subfolder per year is expected
4. The following IPEDS survey components are used:

| Component | File prefix | Description |
|---|---|---|
| Institutional Characteristics | HD | Institution directory, sector, control |
| Graduation Rates | GR | 150% graduation rate by grtype |
| Graduation Rates 200% | GR200 | 8-year graduation rates |
| Finance (GASB) | F1 | Public institution finances |
| Finance (FASB NP) | F2 | Private non-profit finances |
| Finance (FASB FP) | F3 | For-profit institution finances |
| 12-Month Enrollment | EFFY | FTE enrollment |
| Fall Enrollment | EF_A | Headcount by level and gender |
| Student Financial Aid | SFA | Pell grant recipients |
| Human Resources | HR | Instructional staff FTE |
| Salaries (instructional) | SAL_IS | Average instructional salaries |
| Salaries (non-instructional) | SAL_NIS | Average non-instructional salaries |
| Completions | C | Degrees and certificates awarded |
| Admissions | ADM | Admissions test scores and rates |
| Institutional Characteristics (IC) | IC | Institutional characteristics detail |

### Downloading College Scorecard data

1. Go to [https://collegescorecard.ed.gov/data/](https://collegescorecard.ed.gov/data/)
2. Download the **MERGED** files for years 2015–2023
3. Place them in `data/raw/college scorecard/`
4. Variables used: `median_earnings_6yr` (`MN_EARN_WNE_P6`), `median_earnings_10yr` (`MN_EARN_WNE_P10`), `loan_repayment_3yr` (`RPY_3YR_RT`), `completion_rate_sc`, `retention_rate_sc`, `pctpell`

## processed/

These files are generated by running the pipeline scripts in order. Do not edit them manually — re-run the relevant script to regenerate.

| File | Generated by | Description |
|---|---|---|
| `master_aligned_raw.csv` | `01_integrate.R` | Merged longitudinal panel with lagged inputs |
| `master_longitudinal.csv` | `01_integrate.R` | Intermediate merge output |
| `analysis_ready.csv` | `02_clean.R` | Cleaned, imputed, standardized dataset (21,875 rows × 79 columns) |
| `analysis_ready_dataset.csv` | `02_clean.R` | Alternate export of the same file |
| `efficiency_scores_full.csv` | `03_efficiency.R` | DEA/SFA scores and tier classifications, all sectors (1,559 institutions) |
| `efficiency_scores_no_forprofit.csv` | `03_efficiency.R` | Same, excluding for-profit institutions (sensitivity analysis) |

## Key data decisions

- **Year 2024 excluded:** 100% missing across all outcomes. `YEAR_MAX = 2023` is set in `01_integrate.R`
- **Sectors 7, 8, 9, 99 excluded:** Sub-2-year institutions with near-total missingness
- **Graduation rate:** Uses IPEDS grtype 3 (all completers within 150%) divided by grtype 2 (bachelor's-seeking cohort). grtype 4 must not be used
- **Enrollment source:** `total_enrollment_ipeds` and `total_fte` from IPEDS only — College Scorecard enrollment was dropped due to a median 25% discrepancy vs. IPEDS headcount
- **Pell grant source:** IPEDS `pell_pct` only — Scorecard `pctpell` dropped after cross-check
- **Outcome variables are never imputed** — only feature variables are imputed per the missing data policy in `02_clean.R`
