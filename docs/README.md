# docs/

This folder contains documentation generated during the data pipeline — a data dictionary and a set of data quality flag files. These files are for human review and reference only. No pipeline script reads from this folder; nothing here should be edited manually.

---

## Files

### `data_dictionary.csv`
Generated by `02_clean.R`. Documents every column in `data/processed/analysis_ready.csv`, including:
- Variable name
- Source (IPEDS component or College Scorecard)
- Description
- Data type
- Units / scale
- Missingness rate
- Imputation method applied (MICE, median, or none)

Reference this file when interpreting model outputs or writing the methods section.

---

## Flags/

Generated by `01_integrate.R` during the integration step. These files flag potential data quality issues discovered during the merge process. **Review these before proceeding to `02_clean.R`** — they document issues that may require judgment calls.

| File | Description |
|---|---|
| `missingness_summary.csv` | Overall missingness rate per variable across the full panel |
| `missingness_by_year.csv` | Missingness rates broken down by survey year |
| `missingness_by_sector.csv` | Missingness rates broken down by institution sector |
| `opeid_conflicts.csv` | Institutions where OPEID changed across years (251 cases — all legitimate mergers/rebrandings; no action required since joins are on `unitid + year`) |
| `pell_discrepancies.csv` | Institutions where IPEDS and Scorecard Pell% differ by more than 5 percentage points |
| `expenditure_outliers.csv` | Institutions flagged as outliers on expenditure variables before winsorization (notably Strayer University at $100k–$193k/student on `stud_serv_exp_per_fte`) |
| `degree_level_changes.csv` | Institutions that changed degree level (e.g., 2-yr to 4-yr) during the panel period |
| `duplicate_records.csv` | Duplicate institution-year records detected before deduplication |
| `enrollment_discrepancies.csv` | Institutions where IPEDS and Scorecard enrollment differ substantially |
| `scorecard_unmatched_by_year.csv` | College Scorecard records that did not match any IPEDS institution by year |

---

## Notes

- Flag files document conditions at the time of the pipeline run — they reflect the raw data, not the cleaned data
- The `opeid_conflicts.csv` file lists 251 institutions with OPEID changes; these are all verified as legitimate institutional mergers or rebrandings and require no correction
- `expenditure_outliers.csv` informed the decision to winsorize `stud_serv_exp_per_fte` at the 99th percentile rather than the standard threshold used for other variables
