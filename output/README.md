# output/

This folder contains all analytical outputs from the pipeline — trained models, performance metrics, SHAP results, PDP plots, efficiency scores, DEA-ML agreement statistics, and presentation figures. All files here are generated by pipeline scripts and can be fully reproduced by re-running scripts 03–07 from the project root.

---

## Subfolder Structure

```
output/
├── efficiency/         # DEA/SFA results (03_efficiency.R)
├── models/             # Trained models, metrics, predictions (04_ml_models.py)
├── shap/               # SHAP importance values and visualizations (05_shap_interpretability.py)
├── pdp/                # Partial dependence plots and hypothesis test (06_pdp_hypothesis.py)
├── agreement/          # DEA-ML agreement analysis (07_dea_ml_agreement.py)
└── slides/             # Presentation figures (slide*.py scripts)
```

---

## efficiency/

Generated by `03_efficiency.R`.

| File | Description |
|---|---|
| `tier_profiles_full.csv` | Mean values of key variables by efficiency tier (High/Mid/Low), all sectors |
| `tier_profiles_no_forprofit.csv` | Same, excluding for-profit institutions (sensitivity analysis) |
| `efficiency_plots_full.pdf` | Diagnostic plots: score distributions, tier comparisons, DEA frontier |
| `efficiency_plots_no_forprofit.pdf` | Same for the no-for-profit sensitivity run |

Efficiency scores themselves are saved to `data/processed/efficiency_scores_full.csv` (not here) because they are used as inputs by downstream scripts.

## models/

Generated by `04_ml_models.py`.

| File | Description |
|---|---|
| `metrics.csv` | R², RMSE, MAE, MAPE for each model × outcome × split (train/val/test) |
| `random_forest_<outcome>.joblib` | Trained Random Forest pipeline (4 files, one per outcome) |
| `xgboost_<outcome>.joblib` | Trained XGBoost pipeline (4 files) |
| `elastic_net_<outcome>.joblib` | Trained Elastic Net pipeline with StandardScaler (4 files) |
| `predictions_<outcome>.csv` | Test-set predictions with `test_index`, RF, XGBoost, and Elastic Net columns (4 files) |

**Outcomes modeled:** `grad_rate_150_4yr`, `median_earnings_6yr`, `loan_repayment_3yr`, `avg_efficiency_score`

**Key test-set R² values:**

| Outcome | Random Forest | XGBoost | Elastic Net |
|---|---|---|---|
| Graduation rate (6-yr) | 0.806 | 0.810 | 0.635 |
| Median earnings (6-yr) | 0.740 | 0.762 | 0.660 |
| Loan repayment (3-yr) | 0.749 | 0.743 | 0.723 |
| DEA efficiency score | 0.733 | 0.758 | 0.435 |

## shap/

Generated by `05_shap_interpretability.py`.

| File/Folder | Description |
|---|---|
| `shap_heatmap.png` | Cross-outcome heatmap of top feature importance rankings across all models |
| `spearman_shap_vs_permutation.csv` | Spearman ρ between SHAP rankings and permutation importance per model/outcome |
| `<outcome>/global_importance_<model>.png` | Bar chart of mean absolute SHAP values, top features |
| `<outcome>/beeswarm_<model>.png` | Beeswarm plot showing SHAP value distribution per feature |
| `<outcome>/waterfall_high_<model>.png` | Waterfall plot for the highest-predicted institution |
| `<outcome>/waterfall_low_<model>.png` | Waterfall plot for the lowest-predicted institution |
| `<outcome>/permutation_vs_shap_<model>.csv` | Ranked feature table with both SHAP and permutation importance values |

SHAP validation Spearman ρ ranges from 0.795 (loan repayment RF) to 1.000 (efficiency Elastic Net). All p < 0.001.

## pdp/

Generated by `06_pdp_hypothesis.py`.

| File/Folder | Description |
|---|---|
| `hypothesis/hypothesis_test_results.csv` | Full vs. cost-only model R², improvement, and p-value for each outcome |
| `hypothesis/baseline_vs_full_comparison.png` | Visual comparison of full vs. cost-only model performance |
| `<outcome>/pdp_ice_<feature>_random_forest.png` | 1-way PDP + ICE plot for a given feature/outcome |
| `<outcome>/pdp_2way_<feat1>_<feat2>_random_forest.png` | 2-way interaction PDP for top feature pairs |

**Hypothesis test results** (500-permutation test, full model vs. cost-only baseline):

| Outcome | Δ R² | p-value |
|---|---|---|
| Graduation rate | +0.386 | < 0.002 |
| Median earnings | +0.203 | < 0.002 |
| Loan repayment | +0.286 | < 0.002 |
| DEA efficiency | +0.530 | < 0.002 |

## agreement/

Generated by `07_dea_ml_agreement.py`.

| File | Description |
|---|---|
| `agreement_summary.csv` | Spearman ρ and Cohen's κ for RF and XGBoost vs. DEA tiers (n=1,194) |
| `confusion_matrix.csv` | Cross-tabulation of DEA tiers vs. ML-derived quintile tiers |
| `divergent_institutions.csv` | Profiles of 20 institutions where DEA and ML diverge by two full tiers |
| `tier_vs_quintile_heatmap.png` | Heatmap visualization of the confusion matrix |

**Summary statistics:** Spearman ρ = 0.884 (RF), 0.889 (XGBoost); Cohen's κ = 0.600 (RF), 0.605 (XGBoost). Both p < 0.001.
